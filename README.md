# Проект: Мультиклассовое прогнозирование исходов цирроза печени
## [Multi-Class Prediction of Cirrhosis Outcomes](https://www.kaggle.com/competitions/playground-series-s3e26/data?select=train.csv)

### Описание проекта
Данный проект направлен на разработку модели машинного обучения для прогнозирования различных исходов у пациентов с циррозом печени. Модель будет классифицировать пациентов по нескольким категориям исходов заболевания и помогать врачам принимать более обоснованные клинические решения.

Наш проект поможет **выявлять риски летального исхода на ранней стадии** и **улучшать выживаемость пациентов** за счет своевременных медицинских вмешательств. *(мб что-то еще)*

### Цель проекта
Разработать точную модель мультиклассовой классификации исходов цирроза печени.

### Задачи проекта
1. Провести обработку данных и EDA.
2. Выполнить feature engineering *(если нужно будет)*
3. Обучить несколько моделей и сравнить результаты их обучения.
4. Представить отчет о проделанной работе.
